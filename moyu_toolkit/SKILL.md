# 墨羽-自适应记忆工具包

> 给你的 AI Agent 装上 15 大记忆能力，跨会话真正记得你是谁。零平台绑定，开箱即用。

GitHub 仓库：https://github.com/awchzh/moyu-memory

## 15 大能力 — 按领域分组

### 📦 记忆层 — 存储与检索

| # | 能力 | 说明 |
|---|------|------|
| 1️⃣ | **TEMPR 多策略检索** | 语义 + BM25 关键词 + 时间加权，三重保障 |
| 2️⃣ | **工作记忆** | 独立文件存储，上下文压缩也不丢；内建工作流知识库检索（`moyu kb search`） |
| 3️⃣ | **轻量知识图谱** | 自动提取实体关系。基础模式（正则，无需 API），配置 key 后语义深度提取 |
| 4️⃣ | **用户画像** | 自动从对话中提取偏好和事实。基础模式（正则，无需 API），配置 key 后语义深度提取 |
| 5️⃣ | **防重复** | SHA256 哈希去重，不会重复存储 |

### 🧠 学习层 — 从互动中进化

| # | 能力 | 说明 |
|---|------|------|
| 6️⃣ | **从纠正中学习** | 识别"不对/不要/记住"信号，3次晋升为永久规则，同类错误不再犯 |
| 7️⃣ | **自我反思** | 醒来时自动分析旧记忆，发现关联与矛盾 |

### 🛡️ 防御层 — 保护与验证

| # | 能力 | 说明 |
|---|------|------|
| 8️⃣ | **完整性校验** | `moyu check` 手动执行，通过 SHA256 对比检测记忆文件是否被篡改 |
| 9️⃣ | **自动备份与恢复** | `moyu check` 校验通过后自动备份当日文件状态，保留最近 3 天版本。检测到篡改时从备份恢复 |
| 🔟 | **法医分析** | 检测篡改文件中是否包含指令覆盖/角色改写/规则注入等攻击关键词（50+ 中英文模式）及 JSON 结构异常 |
| 1️⃣1️⃣ | **记忆自我保护** | 密码验证。设密码后，删除文件、改配置等危险操作需密码确认。3 次失败自动锁定 30 分钟 |

### 🌿 生命周期层 — 让记忆会呼吸（V2.0.5）

| # | 能力 | 说明 |
|---|------|------|
| 1️⃣2️⃣ | **上下文自适应压缩** | 占用率超过 90% 自动静默压缩——降级低优先级项、截断长记忆、节省 token。支持手动触发（`moyu compress --now`）和状态查看（`moyu context`） |
| 1️⃣3️⃣ | **遗忘曲线（两阶段闸门）** | 十四天安全窗口内不解降级 + 访问密度趋势分析——间隔稳定的保留，间隔扩大的降级。只在上下文有压力时才触发。支持自定义参数：`moyu forget set demote_days 21` |
| 1️⃣4️⃣ | **记忆合并** | 检测关键词重叠的相关记忆，自动合并为一条综合摘要。原始细节存入可展开字段，不丢失任何信息 |
| 1️⃣5️⃣ | **自动更新** | 自动检测 GitHub 新版本，一键升级工具包——保留 memory_data 和用户配置。`moyu update` 检查，`moyu update now` 更新 |

## 快速开始

```bash
pip install numpy requests pyyaml
```

把 `moyu_toolkit/` 文件夹复制到你的项目中：

```bash
cd moyu_toolkit
python3 agent_memory.py search "我们聊过什么"
```

**一条命令统领全局：**

```bash
cd moyu_toolkit && python3 moyu.py help
```

搜索、统计、压缩、遗忘、合并、更新——一个命令搞定。

## 命令参考

所有功能通过统一 CLI 入口使用：

```bash
cd moyu_toolkit && python3 moyu.py <命令> [参数]
```

### 🛡️ 安全

| 命令 | 说明 |
|------|------|
| `moyu setup` | 设置安全密码（危险操作需要密码确认） |
| `moyu verify <类型> [描述]` | 验证危险操作（删文件、改配置等） |
| `moyu unlock` | 解锁安全系统（输错3次自动锁定30分钟） |
| `moyu check` | 完整性校验（SHA256 比对文件） |
| `moyu audit` | 安全体检——三层防御链汇总 |
| `moyu init` | 初始化完整性校验清单 |

### 🧠 记忆与检索

| 命令 | 说明 |
|------|------|
| `moyu search <关键词>` | TEMPR 多策略检索 |
| `moyu stats` | 全统计（记忆/上下文/学习/安全） |
| `moyu status` | 系统状态 + 防御链可视化 |
| `moyu inject` | 获取注入行为规则 |
| `moyu signals` | 查看学习触发词 |

### ⏳ 生命周期与压缩

| 命令 | 说明 |
|------|------|
| `moyu forget` | 遗忘曲线状态（两阶段闸门 + 密度分析） |
| `moyu forget config` | 查看当前遗忘曲线参数 |
| `moyu forget set <键> <值>` | 设置参数：`demote_days`、`archive_days`、`density_window`、`enabled` |
| `moyu forget --summary` | 一行摘要 |
| `moyu compress` | 查看上下文占用率 |
| `moyu compress --now` | 手动触发压缩 |
| `moyu context` | 上下文占用率一行概览 |

### 🔄 学习与反思

| 命令 | 说明 |
|------|------|
| `moyu learn <文本>` | 从纠正中学习 |
| `moyu detect <文本>` | 检测纠正信号 |
| `moyu reflect` | 自我反思（关联与矛盾分析） |

### 📚 知识库

| 命令 | 说明 |
|------|------|
| `moyu kb list` | 知识文件列表 |
| `moyu kb search <关键词>` | 搜索知识 |
| `moyu kb index` | 重建索引 |
| `moyu kb read <文件>` | 读文件 |
| `moyu kb help` | 帮助 |

### 🔗 会话与更新

| 命令 | 说明 |
|------|------|
| `moyu bridge` | 查看会话桥状态 |
| `moyu update` | 检查 GitHub 新版本 |
| `moyu update now` | 下载并更新 |
